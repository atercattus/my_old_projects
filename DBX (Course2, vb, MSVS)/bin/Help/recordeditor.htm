Редактор полей  
Назад  Вверх  Вперед

 
Для удобства редактирования записей в БД в программу введен редактор полей, позволяющий редактировать поля записей более удобным способом, нежели непосредственно в таблице.  
 
 
 
В списке полей отображается список всех полей выбранной в таблице ( двойной щелчок мыши по строке записи ) записи.  
 
Ниже выводится информацио о типе поля: строка либо число.  
 
Кнопка Вернуть восстанавливает значения полей из БД.  
 
В текстовом поле можно редактировать записи.  
 
Редактор используется для редактирования длинных и/или многострочных строк.  
 
Кнопка Применить устанавливает значение выбранного поля во введенное значение.  
 
Кнопка Принять сохраняет запись в БД.  
 
Для возврата в основную программу нажмите Отмена.
