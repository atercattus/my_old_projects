Основы работы с программой  
Назад  Вверх  Вперед

 
Чтобы начать работу с программой практически не требуется начальных знаний. Однако, если Вы хотите уверенно себя чувствовать при работе с программой, рекомендуется сначала почитать эту справку. Здесь будут изложены возможности и методы работы с программой.  
 
Вначале рекомендуется изучить интерфейс программы, чтобы разобраться во всех этих кнопочках и менюшках :), а уже потом переходить к серьезной работе с DB Xtension.  
 
После этого можно перейти к разделу Мастеров и научиться создавать запросы, строить диаграммы и настраивать защиту БД.  
 
И, наконец, можно прочитать о формате запросов для ручной их правки.  
 

 
Для создания новой БД, сохранения, загрузки, создания резервной копии и закрытия используйте одноименные пункты меню Файл, либо кнопки на панели инструментов.  
 
Создание новой БД приводит к закрытию всех таблиц, и созданию пустой таблицы на закладке "Главная таблица".  
При сохранении необходимо выбрать файл, в который будет произведено сохранение АКТИВНОЙ таблицы. Все изменения будут сохранены и путь к текущей БД изменится.  
 
В случае создания резервной копии файл Бд будет скопирован в указанное место, т.е. никакие изменения не сохраняются и текущий путь не меняется.  
 
Закрытие БД приведет к закрытию ВСЕХ таблиц (для закрытия только одной воспользуйтесь контекстным меню, вызываемом щелчко левой кнопки мыши с зажатым [Ctrl] по закладке закрываемой таблицы). Если таблица имеет несохраненные изменения будет выдано сообщение.  
 
При открытии БД сначала вызывается Закрытие (пред. пункт), затем уже происходит загрузка таблицы.  
 

 
Мастер Запросов вызывается пунктом меню Запросы->Мастер запросов. работа с ним описана в разделе Мастер запросов.  
 
Пункт меню Результаты содержит пункты Мастер диаграмм (построение диаграмм по вводимым данным) и Создание HTML.  
Создаваемый HTML-файл формируется на основе данных активной таблицы. Заголовок создаваемой таблицы вводится после указания пути к создаваемому файлу.  
 
Пункт меню Настройки содержит единственный пункт Защита, вызывающий Мастер защиты.  
 
Через пункт ? можно запустить эту справку либо открыть окно О программе.
