Мастер запросов  
Назад  Вверх  Вперед

 
Все операции с данными в таблице осуществляются путем последовательного выполнения запросов. Программа позволяет работать с запросами следующих типов и подтипов:  
 
1.Добавление  
·Добавление нового поля  
·Добавление новой записи  
2.Удаление  
·Удаление поля  
·Удаление записи  
3.Сортировка  
·Сортировка по алфавиту ( для чисел используется математическое сравнение )  
·Сортировка против алфавита  
4.Вывод  
·Вывод записей с совпадающим полем  
·Вывод записей с полем большим заданного  
·Вывод записей с полем меньшим заданного  
·Вывод записей с количеством повторов поля равным некоторому количеству  
·Вывод записей с количеством повторов поля большим некоторого количества  
·Вывод записей с количеством повторов поля меньшим некоторого количества  
5.Обмен  
·Перестановка двух полей  
·Перестановка двух записей  
6.Смена  
·Изменение типа поля ( строка <-> число )  
·Изменение заголовка поля ( названия )  
 
Все эти запросы формируются с помощью диалогов, тем самым делая задачу работы с БД простым и понятным делом.  
 
 
 
Списки Тип запроса и Подпит запроса позволяют выбрать необходимый запрос из всех возможных. Т.к. идеологией работы DB Xtension является блочное выполнение запросов, то создаваемые запросы накапливаются в поле Список подготовленных запросов. Это позволяет изменить или удалить некоторые запросы уже после их создания.  
Для добавления запроса, после выбота его типа, нажмите кнопку , расположенную справа оп списка запросов.  
Для удаления созданного ранее запроса необходимо выбрать его в списке запросов и нажать кнопку .  
Для удаления всех запросов нажмите .  
 
Любой запрос можно редактировать вручную. Если вы уже разобрались с форматом запросов, вы можете редактировать запросы:  
·двойной щелчок в списке запросов выводит запрос в нижнее поле редактирования.  
·после изменения запроса нажмите кнопку справа со стрелкой вверх или [Enter]. В зависимости от того, была ли зажата [Shift] результат может отличаться:  
1.   без зажатой клавиши введеный текст ЗАМЕНИТ выбранный элемент.  
2.с нажатой клавишей или в случае пустого списка подготовленных запросов текст будет ДОБАВЛЕН В КОНЕЦ списка.  
 
Описание формата запросов содержится в разделе Формат запросов.
